> Interactive product grid component with draggable products, zoom controls, and product detail modal.

## Features

- **Draggable Products**: Move products by dragging
- **Zoom Toggle**: Zoom in/out view
- **Product Details**: Click products to view full information
- **Smooth Animations**: GSAP-powered animations

## Tech Stack

- **GSAP** (animations)
- **Draggable.js** (drag functionality)
- **JavaScript (ES6+)**
- **SCSS**
